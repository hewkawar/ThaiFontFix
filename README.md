1. ดาวน์โหลด ThaiFontFix
2. นำไฟล์ .zip ไปใส่ในไดเรคทอรี่ที่ติดตั้งเกม เช่น .minecraft\resourcepacks
3. เปิดใช้งาน ThaiFontFix ในตัวเกม
